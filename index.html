Using Map Method

        const array1=[23, 45, 67, 78, 91];
        const array2=array1.map((x)=>(x+10))
        console.log(array2)

    output:- 
    [33,55,77,88,101]

Using Filter Method

        const arrayfilter=[23,45,67,78,91];
        const arrayfilter2=arrayfilter.filter((x)=>(x<65))
        console.log(arrayfilter2)

    output:- 
    [23,45]

Reduce Method

        const arrayReducer=[23,45,67,78,91];
        const arrayReducer2=arrayReducer.reduce((acc,curr)=>acc+curr);
        console.log(arrayReducer2)
    output:- 
    304

Objects

Objects can be created in three ways:

Object Literal 

        const obj={
            name:"manikanta",
            id:20,
            college:"pragati"
        }

Object Instance

        const obj2=new Object();
        obj2.name="manikanta";
        obj2.id="20";
        obj2.location="secunderabad";
        console.log(obj2)

        Output:-
        {name: 'manikanta', id: '20', location: 'secunderabad'}
        id: "20"
        location: "secunderabad"
        name: "manikanta"

Getting Key and value pairs separately in arrays

    const data={
        name:"joseph",
        age:22,
        salary:"20k"
    }
    
    
    var data1=[];
    var data2=[]
    for (let x in data) {
       data1.push(x);
       data2.push(data[x]);
    }
    
    console.log(data1);
    console.log(data2);
    output:- 
    ['name', 'age', 'salary']
    ['joseph', 22, '20k']

Call,Apply,Bind Methods

Call,apply,bind Methods are used to borrow the functions... 

    const Objects={
        name:"manikanta",
        college:"pragati",
        Function1:function (id,location1){
            console.log(`iam ${this.name} id number is ${id} studies in ${this.college} engineering  college loacated in ${location1}`)

    Objects.Function1.call(Objects,1214,"hyderabad")
